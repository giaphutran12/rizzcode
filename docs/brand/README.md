# RizzCode Brand Assets

The selected Rizz Meter board is the visual source for brand system v1.

## Canonical source

- `source/rizzcode-meter-brand-board.png`: the user-selected Generated image 3
  board. Preserve this file unchanged.

## Production assets

Generated by `npm run brand:assets`:

- `/public/brand/rizzcode-lockup.png`
- `/public/brand/rizzcode-lockup-inverse.png`
- `/public/brand/rizzcode-mark.png`
- `/public/brand/rizzcode-mark-inverse.png`
- `/public/brand/rizzcode-meter-wordmark.png`
- `/public/brand/rizzcode-brand-board.webp`

Do not hand-edit generated files. Update
`scripts/build-brand-assets.mjs`, rebuild, then visually inspect every output.

Use the React `BrandLogo` component instead of referencing asset paths directly:

```tsx
import { BrandLogo } from "@/design-system";

<BrandLogo />
<BrandLogo variant="mark" />
<BrandLogo tone="inverse" />
<BrandLogo variant="meter" />
```

## Asset ownership

- Master identity: Rizz Meter.
- Product progression motif: rank chevron.
- Campaign motif: Patch Notes cursor +1.

Only the master identity belongs in persistent product navigation.
